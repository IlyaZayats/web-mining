О курсе

Короткое описание курса (заглушка): чему научимся, какие навыки прокачаем, какие результаты получит студент. Здесь можно указать цели, компетенции, формат оценивания, ссылки на материалы и т. д.

Цели
  • Освоить синтаксис и стандартную библиотеку
  • Понять практики Go в production
  • Научиться писать сервисы и API
Результат
  • Скелет проекта + архитектура
  • Работа с БД, логированием, тестами

Требования к программному обеспечению

  • Go (версия: заглушка)
  • Git
  • IDE/редактор (GoLand / VS Code)
  • Терминал (bash/zsh/powershell)

  • Docker
  • PostgreSQL / SQLite (по теме БД)
  • Make / Taskfile (по желанию)
  • HTTP-клиент: Postman / Insomnia / curl

  • RAM: 8+ GB (лучше 16)
  • CPU: 2+ ядра
  • Диск: 10+ GB свободного места
  • OS: Windows / macOS / Linux

Проверка готовности

  1. Установи Go
  2. Проверь go version
  3. Проверь git --version
  4. Открой любой модуль и внеси правки
