О курсе

Данный курс предназначен для начинающих Golang разработчиков или тех, кто уже имел опыт работы с другими ЯП.

Цели
  • Освоить синтаксис и стандартную библиотеку
  • Понять практики Go в production
  • Научиться писать сервисы и API
Результат
  • Продвинутые навыки в Go
  • Работа с БД, логированием, тестами
  • Проект из реальных бизнес кейсов
